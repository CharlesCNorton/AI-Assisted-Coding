# ByteWatch: The Real-Time Memory Sentinel

ByteWatch is your go-to Python guardian for real-time RAM and VRAM usage. With a sleek interface and night mode, it's never been easier to keep an eye on your system's memory.

## Requirements
- Python 3.11
- psutil
- GPUtil
- tkinter

## Usage
Unleash the sentinel with `python3 bytewatch.py` in your terminal.

## Features
- Real-time RAM and VRAM usage tracking.
- Refreshes every second for up-to-the-moment data.
- Day/Night mode toggle for around-the-clock system monitoring.
