##################################################
################# DataPatterner ##################
##################################################

ABOUT:
------
DataPatterner is a robust Python tool designed to streamline the process of parsing large textual datasets using regular expressions. Whether you're preparing data for machine learning, research, or any other purpose, DataPatterner ensures that your data conforms to desired patterns with ease. Additionally, with built-in pattern testing and intuitive error handling, you can be confident in the quality of your parsed outputs.

FEATURES:
---------
1. Parse textual data using customizable regex patterns.
2. Test regex patterns on sample data for verification.
3. Interactive command-line interface for user-friendly experience.
4. Error handling to handle common issues and guide the user.
5. File overwrite confirmations to prevent accidental data loss.

USAGE:
------
1. Run the program.
2. Follow on-screen instructions to either parse data, test a regex pattern, or exit.
3. For parsing:
   - Provide the input file path.
   - Specify the output file path.
   - Input your regex pattern.
   - Let DataPatterner do the rest!

4. For pattern testing:
   - Provide the sample data file path.
   - Input your regex pattern.
   - View matched results for verification.

REQUIREMENTS:
-------------
- Python 3.x

NOTE:
-----
Always backup your data before processing to ensure no unintentional data loss.
