# DeDupeText

## Description
DeDupeText is a simple yet powerful utility designed to remove duplicate lines from text files. It is especially useful for cleaning up datasets used in fine-tuning language models.

## Features
- GUI for file selection
- Preserves the original order of lines
- Simple and user-friendly

## Requirements
- Python 3.x
- Tkinter (usually comes pre-installed with Python)

## Usage
1. Run `DeDupeText.py`.
2. Follow the on-screen menu to choose an option:
    - `1` to remove duplicates from a file.
    - `2` to exit the program.
3. If you choose to remove duplicates, a file dialog will appear. Select the text file you want to process.
4. DeDupeText will remove any duplicate lines from the selected file.
