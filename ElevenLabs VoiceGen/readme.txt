=============================================
ElevenLabs VoiceGen: Text-to-Speech Converter
=============================================

Description:
------------
ElevenLabs VoiceGen is a Python tool that leverages the ElevenLabs API to convert text into speech. With a variety of voice options available, you can choose your preferred voice and generate audio files on-the-go.

Pre-requisites:
---------------
- Python 3.x
- `requests` library (You can install it via pip: `pip install requests`)

Setup:
------
1. Clone/download the repository.
2. Navigate to the directory containing the program.
3. Open the script and replace `YOUR_API_KEY` with your ElevenLabs API key.
4. Save and close the script.

Usage:
------
1. Run the script using the command: `python voicegen.py` (or the name you saved it as).
2. Follow the on-screen instructions to list voices, convert text to speech, set default output directory, or exit.

Note:
-----
1. Always make sure you have a valid API key and internet connection before using the program.
2. For any issues, refer to the official ElevenLabs API documentation or open an issue on GitHub.
