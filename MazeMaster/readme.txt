=====================
Maze Generator and Solver
=====================

This Python script generates a maze based on user-defined difficulty levels and solves the maze using Dijkstra's algorithm. The size of the maze is determined by the difficulty level chosen.

This script is compatible with Python 3 and requires the following Python libraries:
- matplotlib
- networkx

Follow the on-screen instructions to generate and solve a maze!

For any issues or improvements, please open an issue or submit a pull request on the GitHub repository.

Have fun generating and solving mazes!
