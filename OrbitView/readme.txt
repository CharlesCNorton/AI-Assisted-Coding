--------------------------------------------------------------
README - OrbitView: An Interactive Solar System Explorer
--------------------------------------------------------------

OrbitView is an interactive application developed in Python that allows users to visualize the orbits of planets in our solar system for any given date. 

The application provides a simple and intuitive interface that includes a dropdown menu to select a planet, a date picker to select a date, and a scale to control the viewing range (zoom level).

Usage:
1. Run the OrbitView.py script in a Python environment with the necessary dependencies installed (tkinter, matplotlib, ephem, numpy, tkcalendar).
2. On the application window, choose a planet from the dropdown menu.
3. Select a date from the date picker.
4. Adjust the viewing range using the scale.
5. Click "Get Position" to calculate and display the position of the selected planet on the chosen date and plot the orbits of all planets.
6. You can use the "Clear" button to reset your selections.

Dependencies:
- Python (version 3.6 or higher)
- tkinter
- matplotlib
- ephem
- numpy
- tkcalendar

To install dependencies, use pip:

pip install -r requirements.txt

This will install all necessary packages.
