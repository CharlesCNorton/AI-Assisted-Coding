SafeTensors Converter
=====================

Description:
------------
SafeTensors Converter is a graphical tool designed to facilitate the conversion of .bin files to .safetensors format. It provides an intuitive interface to select directories and perform the conversion seamlessly.

Installation:
-------------
1. Ensure you have Python 3.x installed.
2. Install the required packages:
   - `pip install transformers safetensors torch tkinter`
3. Clone the repository or download the script.

Usage:
------
1. Run the script: `python safe_tensors_converter.py`
2. Use the "Select Directory" button to choose the directory containing the .bin files.
3. Click "Convert" to start the conversion process. The converted files will be saved in the same directory with the .safetensors extension.

Note:
-----
- Ensure that the directory you choose contains valid .bin files for the conversion.
- Make sure to have enough storage space in the chosen directory for the converted files.
