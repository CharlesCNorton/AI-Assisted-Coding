Title: SimpleCalculator with Mathematical Functions and Tooltips

Version: 1.2.0
Date: 2023-07-07

Description:
This program is a highly-enhanced, object-oriented calculator that can perform a range of arithmetic operations, including support for mathematical functions such as sqrt, log, sin, and cos. Each mathematical function and constant is provided with a hover tip to guide the user on its usage and significance.

Dependencies:
- Python 3.x
- tkinter

How to Run:
Simply run the main calculator.py file. This will launch the graphical interface of the calculator. You can then enter your expressions using the buttons on the interface or directly via your keyboard.

Input:
This calculator accepts numbers, parentheses, and the following operators: +, -, *, /, ^. It also supports the following mathematical functions: sqrt, log, sin, cos.

Updates in Version 1.2.0:
1. Added tooltips for mathematical functions and constants.
2. Support for mathematical functions: sqrt, log, sin, and cos.
3. Enhanced security by replacing the eval function with a safer alternative.
