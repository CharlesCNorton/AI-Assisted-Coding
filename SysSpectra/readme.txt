SysSpectra

SysSpectra is a lightweight utility to monitor your system's VRAM and RAM usage. It offers a sleek interface and a night mode for better visibility in low-light conditions.

Installation

Dependencies:

1. psutil
2. GPUtil
3. tkinter (usually comes with Python)

You can install the required modules with:
pip install psutil GPUtil

Usage

Simply run the syspectra.py file:
python syspectra.py

This will display a small window with the current VRAM and RAM usage. You can toggle between day and night modes with the provided button.
