# Tinnitus Relief App

## Description
This is a simple tinnitus relief application built with Python. The application generates a tone that the user can control in terms of frequency and volume, as well as the volume of the white noise. This can be used to match the frequency of the user's tinnitus and help in providing some relief.

## Usage
To use the application, simply run the `tinnitus.py` script with Python. A GUI will open with sliders to control the frequency, volume of the tone, and volume of the white noise. Click the "Turn On" button to start generating the tone, and the "Turn Off" button to stop it.

## Requirements
This application requires Python, with the following packages:
- numpy
- sounddevice
- tkinter
- scipy
