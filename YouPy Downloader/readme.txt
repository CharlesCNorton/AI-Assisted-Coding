YouPy - A Simple YouTube Video Downloader
=========================================

Welcome to YouPy, a user-friendly tool to download your favorite YouTube videos directly to your computer.

Features:
---------
- URL validation to ensure valid YouTube links.
- Directory validation with an option to create if not exists.
- Progress bar that shows the download progress.
- Menu-driven interface for easy navigation.
- Enhanced error handling for a smooth user experience.

Usage:
------
1. Run the YouPy script.
2. Enter the YouTube video URL when prompted.
3. Specify your desired output path.
4. The program will download the video and save it to the specified directory.
5. You can continue downloading more videos or exit the program.
